Confidentialité
TrackMap est conçu pour protéger votre vie privée, pas pour la compromettre.

Ce que TrackMap collecte

  • Aucune donnée personnelle n'est collectée
  • Aucun serveur distant n'est contacté
  • Toute l'analyse se fait localement dans votre navigateur
  • Les données de session sont effacées à la désinstallation
  • Les options sont synchronisées via le compte Chrome/Firefox (chiffré de bout en bout)
100% local · Zéro télémétrie · Open source

Données stockées localement

TrackMap stocke dans chrome.storage.local : le graphe de navigation de la session en cours (domaines visités, arêtes, positions), les données de la page active (trackers détectés, scores), et les positions des nœuds du graphe pour restaurer la vue. Ces données ne quittent jamais votre appareil.

Les options (sites exclus, niveau de détection, préférences d'affichage) sont stockées dans chrome.storage.sync et synchronisées via votre compte de navigateur si vous êtes connecté.

À la désinstallation

Lorsque vous désinstallez TrackMap, toutes les données locales (chrome.storage.local et chrome.storage.sync) sont effacées automatiquement via l'événement chrome.runtime.setUninstallURL.
