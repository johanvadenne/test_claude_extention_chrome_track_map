TrackMap

Politique de confidentialité

Dernière mise à jour : mars 2026 · Version 5.0

TrackMap est une extension de navigateur conçue pour vous informer sur les trackers présents sur les sites que vous visitez. Elle ne collecte, ne transmet et ne vend aucune de vos données personnelles.

Toute l'analyse se fait localement dans votre navigateur
Aucun serveur distant n'est contacté lors de l'analyse
Aucune donnée de navigation ne quitte votre appareil
Les données locales sont effacées à la désinstallation
Code source auditable, aucune télémétrie

1. Ce que TrackMap ne collecte jamais

Les éléments suivants ne sont jamais collectés, stockés ou transmis :

Votre identité, nom, adresse e-mail ou tout autre identifiant personnel
L'historique complet de votre navigation (seule la session en cours est analysée)
Le contenu des pages web que vous visitez
Vos mots de passe, données de formulaire ou informations de paiement
Des données de localisation ou de l'appareil à des fins de profilage
Des statistiques d'utilisation ou de télémétrie

2. Données stockées localement

TrackMap stocke les données suivantes uniquement sur votre appareil, jamais sur un serveur externe :

Emplacement Contenu Durée
chrome.storage.local Graphe de session (domaines visités, connexions, positions des nœuds), données de la page active (trackers détectés, scores de risque) Session en cours, effacé à la désinstallation
chrome.storage.sync Options utilisateur uniquement : sites exclus, niveau de détection, préférences d'affichage. Aucune donnée de navigation. Jusqu'à suppression manuelle ou désinstallation

Les données dans chrome.storage.sync sont synchronisées par le navigateur entre vos appareils si vous êtes connecté à votre compte Chrome ou Firefox. Cette synchronisation est chiffrée de bout en bout par Google/Mozilla et TrackMap n'y a pas accès.

3. Accès réseau

TrackMap accède au réseau uniquement pour :

Chargement des polices Google Fonts dans les pages d'options et de confidentialité. Ces requêtes ne contiennent aucune donnée de navigation et peuvent être évitées en utilisant une version hors-ligne.

TrackMap n'effectue aucun appel réseau pendant l'analyse des pages que vous visitez. La base de données de trackers (trackers-db-full.json) est entièrement incluse dans le bundle de l'extension et consultée localement.

4. Permissions utilisées

PermissionUsage
tabsIdentifier l'onglet actif pour afficher les données correspondantes dans le popup.
storagePersister le graphe de session et les options localement.
scriptingInjecter le content script dans les pages HTTP/HTTPS pour détecter les scripts tiers.
webRequestIntercepter les requêtes réseau pour détecter les trackers chargés dynamiquement. Aucune requête n'est modifiée ou bloquée.
cookiesRéservé pour une future fonctionnalité d'analyse des cookies tiers. Non utilisé dans la version actuelle.
host_permissions (http, https)Nécessaire pour que webRequest puisse observer les requêtes sur toutes les pages.

5. Désinstallation et suppression des données

Lors de la désinstallation de TrackMap, l'extension efface automatiquement toutes les données locales (chrome.storage.local) via l'événement chrome.runtime.onInstalled.

Les données chrome.storage.sync (options utilisateur) peuvent être effacées manuellement depuis la page Options → Confidentialité → "Effacer toutes les données maintenant", ou automatiquement à la désinstallation selon la politique du navigateur.

6. Tiers et services externes

TrackMap utilise les listes de trackers publiques suivantes pour construire sa base de données, consultées au moment du build uniquement (pas à l'exécution) :

Disconnect.me Tracker Protection List (licence MPL 2.0)
DuckDuckGo Tracker Radar (licence Creative Commons)
EasyPrivacy (licence CC BY-SA 3.0)

Ces listes ne sont pas consultées lors de l'utilisation de l'extension — elles sont intégrées dans le bundle à la compilation.

7. Modifications de cette politique

Toute modification de cette politique de confidentialité sera accompagnée d'une mise à jour de la version de l'extension visible dans la page À propos. Les modifications substantielles feront l'objet d'une notification dans le popup au premier lancement après mise à jour.
